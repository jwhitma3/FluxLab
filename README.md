# FluxLab

> [!NOTE]
> **Work in Progress:** This project is in development and is far from a functional Minimum Viable Product. For now, expect frequent bugs, breaking changes, and incomplete features. I am working on this project independently and on a time-available basis. Any updates will be infrequent.


# Overview
`FluxLab` is Python GUI tool for measuring emission line fluxes, calculating ionic and elemental abundances, and conducting other spectral analysis of planetary nebulae. FluxLab is designed to replace an existing CLI-based IDL pipeline, giving users an intuitive, visual way to plot spectra and analyze observational data.

# Project Status and Roadmap

## Interactive Spectrum Plotting
- [ ] Load 1D spectrum from FITS or ASCII files
- [ ] Panning and zooming tools to smoothly navigate data.

## Measurement tools
- [ ] Simple direct integration
- [ ] Manual and automatic continuum placement
- [ ] Systematic uncertainty of continuum placement
- [ ] Single and multiple component gaussian fits

## Analysis Tools
- [ ] NIST and Atomic Line List integration for visual line identification
- [ ] PyNeb integration
